## How it works

The GitHub Action workflow automatically:
1. Scans all PDF files starting with `Exam_Result_Sheet_dsce_sem`
2. Extracts SGPA values from each PDF
3. Calculates the CGPA (average of all SGPA values)
4. Updates this README with the current CGPA
